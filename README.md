# Logs – Digital Wellbeing

Logs is a privacy-first digital wellbeing app that helps you understand your digital habits, stay focused, and reduce distractions across your devices.

This repository is used to track issues, bugs, and feature requests for the Logs desktop app and browser extensions.

## Report an issue

If you encounter a problem or have a suggestion:

1. Open the **Issues** tab
2. Click **New issue**
3. Choose Bug or Feature request
4. Describe the problem or idea

Please include:
- Logs version
- Windows version
- Browser (if extension issue)
- Steps to reproduce (for bugs)

## About Logs

Logs provides:
- App and website usage insights
- Focus sessions and distraction blocking
- Cross-device digital wellbeing
- Privacy-first local data
